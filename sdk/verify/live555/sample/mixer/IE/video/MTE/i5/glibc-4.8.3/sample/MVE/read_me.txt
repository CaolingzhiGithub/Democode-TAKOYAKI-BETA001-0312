1. Lib is in folder lib
   a. libMVE_LINUX.a is static version
   b. libMVE_LINUX.so is dynamic version

2. Header files are in folder include

3. Sample code and images are in folder sample
   a. sample_main.c is the entry of sample code
   b. command "make" can build the sample code
   c. two execution files are generated after make
      run_linux_static: static version
	  run_linux_dynamic: dynamic version
